KeT-LMSの使い方           2023-09-20

  • 準備
    • Cinderellaのインストール
       KeTCindy Home 内「最新ニュース」の「Cinderellaのインストール」
       を参考にインストールする
    • KeTCindyのインストール
       KeTCindy Home 内「KeTCindyとは」の「KeTCindyのインストール」
       を参考にインストールする
    • Maximaのインストール
       KeTCindy Home 内「最新ニュース」の「Maximaのインストール」
       を参考にインストールする
    • KeT-LMSのファイル一式のダウンロード
      1.  ketmath の右にある「Releases/Latest」からダウンロードする
      2.  ダウンロードしたzipを解凍する
      3.  適当なフォルダに移動し，名称を変える
  • 基本編
     問題の作成から答案の返却までの手順です
     （以下のファイル名，フォルダ名はダウンロードしたフォルダ内のもの）
    1. クラス名簿作成
      • フォルダ data 内の student2023.txt のように作成する
      • 名簿ファイルは data 内に置く
      • 複数のクラス名簿を扱うことできる
        • 複数のときは，以下の手順で，toolkemath.cdyのdataボタンでクラスを選択して行う
    2. 課題のテキストファイル作成
      • KeT-LMSの課題ファイル を参考にする
      • フォルダ data 内の question001-1.txt のように作成する
      • 課題ファイルは data 内に置く
        （以下の手順で作られるファイルの名称は question001-1.txt からのものである）
    3. 課題のHTML作成
      1.  toolkemath.cdyを立ち上げる
      2. 「1. taskline を作成」を押し，「実行」を押す
        3.   $\to$ data 内に 1taskline 001-1.txt , 2anssheet 001-1.txt , 2anssheet 001all.txt が作られる
      3. 「2. kettask に組込」を押し，kettaskorgv を選択し「実行」を押す
        4.   $\to$ 課題 kettaskv 001-1.html が作られる
    4. 課題の配布
      5.  課題を自分のホームページなどに置き，そのURLを学生に伝える
    5. 学生の解答
      6.  学生は画面上のキーボードを使って入力する
      1. 課題のHTMLを開く
      2. 自分の番号を入力し，「OK」を押す
      3. もう一度「OK」を押すと，大問が表示される
      4. 左の「>」を押すと，小問が表示される
        5.   $\to$ 問題に解答する
      5. もう一度「>」を押すと，次の小問が表示される
        6.  「<」を押して，前の小問を解答し直してもよい
      6. すべての問題に解答後，左下の「Rec」を押す
        7.   $\to$ 下の枠内に表示されるテキストをすべて選択コピーし，教員に送る
    6. 採点
      1.  toolkemath.cdy を立ち上げる
      2.  回収した答案データを data 内の 2anssheet 001all.txt に貼り付ける
      3. 「3. anssheetall 分割」を押し，「実行」を押す
        4.   $\to$ 問題ごとの 2anssheet 001-x.txt にコピーされる（1題のときは1つ）
      4. 「4. scoreline+scorechart」を押し，「実行」を押す
        5.   $\to$ data 内に 3scoreline 001-1.txt, 4scoresheet 001-1.txt, 4scoresheet 001all.txt が作られる
      5. 「5. ketscore 組込」を押し，ketscoreorgvn を選択し「実行」を押す
        6.   $\to$ ketscorev 001-1n.html が作られる
      6. 手動で採点するときは，手動の採点 を参考にする
        7.    Maximaで採点するとき
           「6. Maxima 採点 option」を押し，「実行」を押す
          $\to$ 4_scoresheet 001-1.txt が作られる
         注）エラーがある場合，errormaxima 001-1.txt を参考にし，3scoreline 001-1.txt を修正する
      7.  i.  ketscorev001-1n.html を開く
           ii.  4_scoresheet 001-1.txt の中身をコピーし， ketscorev001-1n.html の最下欄に貼り付ける
             iii.「st」,「pg」で学生，ページを動かして，点数を確認，修正する
          iv.「Rec」を押し，最下欄のテキストをすべて選択コピーし，4_scoresheet 001-1.txt の
           2行目に貼り付ける
    7. 集計
      1.  toolkemath.cdyを立ち上げる
      2. 「8. 問別個人別成績表作成」を押し，「実行」を押す
        3.   $\to$ data 内に card が作られ，そのフォルダ内に問ごと，学生ごとの成績表が作られる
      3. 「9. 個人別結合表作成」を押し，kettaskorg を選択し「実行」を押す
        4.   $\to$ card 内に，学生ごとの成績表が作られる
      4. 「10. 個人別結合表作成」を押し，kettaskorg を選択し「実行」を押す
        5.   $\to$ フォルダ内に，成績一覧表が作られる
    8. 返却
      1.  Dropboxに返却用のフォルダを作る
      2.  toolkemath.cdyを立ち上げる
      3.  スクリプトエディタの7行目に返却用のフォルダ名を入力する
      4. 「10. 個人別結合表作成」を押し，「実行」を押す
        5.   $\to$ Dropboxに学生ごとのフォルダが作られ，成績表が返却される
